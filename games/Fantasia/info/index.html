Changelog

Jump to Version

▼

Alpha 1

  • Combats
  • Mobile entities
  • Modular quest system
  • savegames

Alpha 1.5

  • A new town north of the farmer's house
  • Many bug fixes
  • Updates to NPC conversation
  • More NPCs
  • Tutorial level
  • More quests
  • More responsive combat

Alpha 1.5.1

  • Updates to the tutorial

Alpha 1.6

  • Fixed a bug that caused NPCs to display the incorrect dialogue.
  • Added more NPCs
  • Added a miniboss

Alpha 1.6.5

  • Fixed some issues with the miniboss fight.
  • Updated one of the NPCs outside of the cave's texture.
  • Probably did some other stuff but I forgot.

Beta 1.6.6

  • Fixed the reset button.
  • Multiple notifications can now display simutaneously.
  • Notification about the spell book now appears at level 3, when the spellbook does; previously, it would appear at level 5.
  • Added interiors to buildings.
